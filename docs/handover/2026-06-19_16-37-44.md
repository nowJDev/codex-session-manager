## 실패했던 접근과 해결

- 처음에는 GitHub CLI가 없어 원격 fork 자동 생성이 막혔다.
- GitHub CLI 설치와 인증 후 fork 상태를 확인했다.
- Tauri updater signing key를 GitHub Actions secret에 저장했지만, Windows PowerShell stdin 경로에서 UTF-8 BOM이 붙어 Actions에서 `Invalid symbol 239, offset 0` 오류가 발생했다.
- `gh secret set -f`와 BOM 없는 임시 dotenv 파일 방식을 사용해 secret을 다시 저장했다.
- 재실행한 `v0.5.4` 릴리즈 워크플로는 성공했다.
