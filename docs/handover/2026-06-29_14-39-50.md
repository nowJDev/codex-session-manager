- 실험적 `/status` 자동 조회는 안정적으로 동작하지 않아 제거했다.
  - Codex TUI를 백그라운드에서 켠 뒤 `/status`를 입력해 파싱하는 방식은 환경/화면/언어/터미널 제약에 취약했다.
  - 현재는 공식 usage 페이지 이동 버튼으로 대체했다.
- README 스크린샷을 처음에는 샘플 렌더 이미지로 만들었으나 실제 앱 우측 하단 패널과 모양이 달라 재작업했다.
  - 최종 스크린샷은 실제 앱 캡처 기반이다.
- PowerShell 이미지 처리 시 UTF-8/코드페이지 이슈가 있었으나, 최종 작업에는 임시 렌더 스크립트를 남기지 않았다.
- `gh release view --json isLatest`는 현재 gh 버전에서 지원하지 않았다.
  - `gh release list`로 `Latest` 상태를 확인했다.
- Release workflow는 태그 push만으로 목록에 바로 보이지 않아 `gh workflow run Release -R nowJDev/codex-session-manager --ref v0.5.7`로 직접 실행했다.
